    MongoDB Shell Enhancements for Hackers

    by TylerBrock

    Installation

    $ git clone git@github.com:TylerBrock/mongo-hacker.git
$ cd mongo-hacker
$ ln -sf mongo_hacker.js ~/.mongorc.js

    Features

    Colorized Shell

    Aggregation Framework Helpers

    Group and Count
    gcount(group_field, filter)
    Group and Sum
    gsum(group_field, sum_field, filter)

    Sensible Defaults

    • Verbose shell enabled
    • Auto multi update
    • Two spaces indent
    • Colon next to keys

    Index Paranoia

    Print index usage information on every query

    Colorize and Highlight Query Time

    Highlights slow running queries
